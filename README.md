Bolo Sport Product Catalog

This project is a front end e commerce website built using HTML, CSS, and JavaScript. It simulates a modern athletic apparel store and was created to understand how product based websites work behind the scenes.

Features
Dynamic product rendering using JavaScript
Easy to add or update products through a structured data file
Image hover effect for multi image products
Product filtering by category
Sorting by price
Clean grid layout inspired by real online stores
How It Works

Products are stored in a JavaScript array with details like name, price, category, and images. The site reads this data and displays each product automatically. This allows the catalog to scale without changing the layout.

Project Structure
index.html handles the layout
styles.css controls the design
script.js manages product data and functionality
Purpose

The goal of this project was to learn how front end e commerce systems function. It focuses on how data connects to what users see, and how small details can improve the overall experience.
