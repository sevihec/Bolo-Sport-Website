Bolo Sport Product Catalog

This project is a front end e commerce website built using HTML, CSS, and JavaScript. It was created to simulate a modern athletic apparel store while learning how product based websites function.

Features
Dynamic product rendering using JavaScript
Easy product updates through a structured data file
Image hover effect for multi image products
Product filtering by category
Sorting by price
Clean grid layout inspired by real online stores
How It Works

Products are stored in a JavaScript array with details like name, price, category, and images. The site reads this data and displays each product automatically, making it easy to scale without changing the layout.

Purpose

This project goes beyond practice. The website is being developed to go live for a real brand, Bolo Sport.

The owner is a competitive athlete in a sport that does not provide much financial support. Because of that, professional web development was not a realistic option. This project gave me the opportunity to build and deliver a fully functional website at no cost, helping support and promote his brand.

At the same time, it allowed me to gain hands on experience building a real world product that connects directly to a business need.
